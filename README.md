# Rubricodex

Rubricodex는 모호한 Codex 구현 요청을 명확한 끝점, evaluation matrix, implementation taskpack, 병렬 probe, evidence 기반 scorecard로 바꾸는 Codex-focused harness입니다.

제품 스펙, 아키텍처, 주요 개념, artifact 계약, v0.1 roadmap은 아래 단일 SSoT 문서에서 관리합니다.

- [docs/rubricodex-ssot.md](docs/rubricodex-ssot.md)

예전 draft, chat transcript, 임시 메모를 제품 기준으로 보지 않습니다. 먼저 SSoT를 수정하고, schema, CLI behavior, example, taskpack은 그 문서에서 파생합니다.
